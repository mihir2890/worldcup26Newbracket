# KickOff 2026 — World Cup Bracket Challenge

A vibrant bracket prediction game for FIFA World Cup 2026.

## Deploy to Netlify (free, 2 minutes)

1. Go to [netlify.com](https://netlify.com) and sign up free
2. Click **"Add new site" → "Deploy manually"**
3. Drag & drop the `wc2026-site` folder onto the page
4. Your site is live instantly! 🎉

## Deploy to Vercel (free, 2 minutes)

1. Go to [vercel.com](https://vercel.com) and sign up free
2. Click **"Add New → Project"**
3. Drag & drop the folder, or use the Vercel CLI:
   ```
   npm i -g vercel
   vercel --cwd wc2026-site
   ```

## Usage

1. Get a free API key at [jsonbin.io](https://jsonbin.io)
2. Open the game, enter your API key, create a league
3. Share the URL + league code with friends
4. Fill in your bracket before June 11!

## Files

- `index.html` — Landing page
- `game.html` — The full bracket game
- `netlify.toml` — Netlify config
